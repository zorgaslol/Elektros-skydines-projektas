# Elektros Skydinės Projektas

Web application for configuring an electrical distribution board using a drag-and-drop interface, automatic price calculation, and configuration export functionality.

---

## Project Overview

This project allows users to visually configure an electrical distribution board (skydinė) by dragging and placing different electrical components into a panel layout. The application dynamically calculates the total price based on selected components and allows saving or exporting the configuration.

The project focuses on practical JavaScript logic and interactive frontend behavior.

---

## Features

- Drag-and-drop interface for electrical components
- Automatic price calculation based on selected items
- Ability to save or export the configuration
- Uses external JSON data for component definitions
- Runs entirely in the browser (no backend required)

1. Open the application in your browser
2. Drag electrical components into the panel area
3. Observe the total price update automatically
4. Save or export the configuration if needed
